🧳 Cassetto_Utility
Toolbox per il Cassetto Fiscale

Userscript per Tampermonkey/Greasemonkey che aggiunge una barra degli strumenti al Cassetto Fiscale dell'Agenzia delle Entrate con download massivo F24/F23 e Report Excel.

Guida rapida

Installazione

Richiede un'estensione per userscript nel browser. L'installazione richiede meno di 2 minuti.

1

Installa l'estensione del browser

Scegli l'estensione per il tuo browser:

Tampermonkey per Chrome/Edge · Tampermonkey per Firefox · Greasemonkey per Firefox

2

Installa lo userscript

Clicca il pulsante qui sotto — Tampermonkey aprirà automaticamente la finestra di conferma:

➤ Installa Cassetto_Utility.user.js

3

Accedi al Cassetto Fiscale

Vai su cassetto.agenziaentrate.gov.it, naviga in Versamenti → Modello F24 oppure Versamenti → Modello F23, seleziona l'anno — la barra verde apparirà automaticamente in cima alla pagina.

⚠️ Consenti sempre i download multipli quando il browser lo chiede per agenziaentrate.gov.it.

Cosa fa

Funzionalità

Tutto avviene client-side — nessun dato viene inviato a server esterni.

🧳⬇

Scarica F24

Download massivo delle quietanze PDF (o Copia F24 come fallback) per tutti i versamenti dell'anno selezionato. Registra i risultati per il Report Excel.

🧳⬇

Scarica F23

Download massivo delle copie PDF dei modelli F23. Nessuna quietanza disponibile per F23 — solo copia del modello.

📊

Report Excel

Genera un file .xls con Riepilogo + Dettaglio. Post-download mostra il raffronto tra lista sito e file scaricati: ✅ scaricato · ❌ errore · ⏳ non scaricato.

📋

Protocolli F24

Copia negli appunti data + numero di protocollo di tutti i versamenti F24 visibili nella pagina.

🔍

Riepilogo

Toggle rapido: mostra o nasconde nella barra i conteggi di versamenti trovati, quietanze disponibili e file scaricati.

🔄

Navigazione dinamica

La barra rileva automaticamente i cambi di pagina e aggiorna i pulsanti al contesto corrente. Dalla pagina Versamenti (Ric=VERS), pulsanti diretti per raggiungere Modello F24 o F23. Non serve mai ricaricare la pagina.

👥

Cassetto delegato

Rilevamento automatico dell'identificativo: PIVA del soggetto delegato, PIVA propria (11 cifre), o Codice Fiscale (16 caratteri) con cascata di priorità.

Per sviluppatori

Note tecniche

Privacy by design

Tutte le elaborazioni avvengono esclusivamente client-side nel tuo browser. Nessun dato viene raccolto, salvato o trasmesso a server esterni. Lo script opera solo nel dominio cassetto.agenziaentrate.gov.it.

Dettagli implementativi

Lo storage usa GM_setValue/GM_getValue (Tampermonkey) con fallback automatico su localStorage. Il monitoraggio URL (polling ogni 500ms) rileva la navigazione interna e ricostruisce i pulsanti della barra senza bisogno di ricaricare la pagina. La quietanza F24 è disponibile solo per versamenti successivi al 01/10/2006 tramite servizi telematici AdE. Il delay tra download è di 600 ms.
